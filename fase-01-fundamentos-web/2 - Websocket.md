Websocket é um protocolo que torna possivel a comunicação entre o navegador web e o servidor
Ele permite a criação de um tunel para transportar de forma continua informações entre o navegador e o servidor de uma aplicação
Ele faz parte das expecificações do HTML5 e é suportadortado pela ultimas versões dos navegadores
Utilizando como exemplo o protocolo HTTP, este realiza uma conexão quando o navegador solicita uma informação, o servidor faz os procedimentos internos e devolve a informação ao navegador e encerra a coenxão em seguida, e uma nova conexão é criada apenas quando há uma nova requisição
No caso do websocket é criada uma conexão persistente onde as informações são enviadas e recebidas de forma continua,sem que sejam feitas várias requisições a cada nova informação que for enviada ou recebida.
É aconcelhado apenas em aplicações que demanda comunicação bidirecional em tempo real
