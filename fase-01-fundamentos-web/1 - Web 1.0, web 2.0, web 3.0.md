🌐 Web 1.0 — A internet original

Foi a primeira fase da web, com sites funcionando principalmente entre 1991 e 2004.

As páginas eram estáticas conteúdo fixo que o usuário apenas lia.

Não havia muita interação: os usuários eram em sua maioria consumidores de conteúdo, não produtores.

Resumo: web de “leitura somente”, simples e informativa.

🌍 Web 2.0 — A web social e interativa

A partir de cerca de 2004, a web passou a permitir interação e colaboração entre usuários.

Surgiram redes sociais, blogs, comentários, vídeos e conteúdo gerado pelos próprios usuários.

É uma web mais “dinâmica”: usuários leem, escrevem, compartilham e interagem.

Plataformas como Facebook e YouTube são exemplos típicos de Web 2.0.

Resumo: web de “leitura e escrita” — mais colaborativa e social.

🔗 Web 3.0 — A próxima geração descentralizada

É a evolução atual/em construção da web, focada em inteligência, descentralização e controle do usuário.

Usa tecnologias como blockchain, contratos inteligentes, inteligência artificial e sistemas distribuídos.

Permite que o usuário controle seus próprios dados e identidade digital em vez de depender de plataformas centralizadas.

A Web 3.0 muitas vezes é chamada de web em que você lê, escreve e possui seus dados.

Resumo: web de “leitura, escrita e propriedade” — mais inteligente e descentralizada.

🧠 Comparação simples
Versão	Usuário	Conteúdo	Controle
Web 1.0	Apenas consome	Estático	Centralizado
Web 2.0	Interage e cria	Dinâmico	Centralizado por plataformas
Web 3.0	Controla e possui dados	Inteligente e personalizado	Descentralizado
📌 Em uma frase:

Web 1.0: web estática e informativa.

Web 2.0: web social e colaborativa.

Web 3.0: web inteligente e descentralizada, dando mais controle ao usuário.
