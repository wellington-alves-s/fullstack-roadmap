# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Purpose

This is a personal full-stack developer learning roadmap tracking progression through structured curriculum phases. Content is written in Brazilian Portuguese. There is no build system, test runner, or CI/CD — it is a study and exercise repository.

## Code Formatting

Prettier is configured (`.prettierrc`):
- Tabs (not spaces), tab width: 4
- Print width: 100 characters
- Double quotes

## Repository Structure

Each phase (`fase-XX-*/`) follows this convention:
- `resumo.md` — study notes and key learnings
- `recursos.md` — useful links and references
- `projetos/` — practical projects

HTML/CSS exercises are organized under `fase-02-front-end/html-css/Modulo N/exercicioNNN/`, each containing an `index.html` plus any supporting assets (fonts, images).

Python work lives in `fase-03-backend/Python/` as Jupyter notebooks (`.ipynb`).
