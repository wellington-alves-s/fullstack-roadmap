Resumo do que foi estudado na pasta html-css

- Estrutura de pastas:
  - arquivos HTML com marcação semântica
  - arquivos CSS para layout, tipografia e responsividade
  - possíveis assets como imagens, ícones e fontes

- Conteúdo principal:
  - Criação de páginas estáticas usando tags HTML semânticas
  - Aplicação de estilos CSS para cores, fontes, espaçamentos e componentes visuais
  - Uso de layout com Flexbox e/ou Grid
  - Implementação de responsividade com media queries
  - Ajustes de acessibilidade básicos e melhores práticas de organização

- Aprendizado:
  - Domínio da estrutura HTML e cascata de CSS
  - Uso de seletores, box model e posicionamento
  - Adaptação do design para diferentes tamanhos de tela
  - Organização do projeto em pastas e arquivos claros
